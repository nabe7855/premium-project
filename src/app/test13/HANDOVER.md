# プロジェクト引き継ぎ・開発再開ガイド：次世代型女性向けコンテンツハブ

本ドキュメントは、プロジェクトの現在の進捗、設計思想、および将来の拡張計画をまとめたものです。どのAIが引き継いでも、即座に開発を継続できるように構成されています。

## 1. プロジェクト概要・ビジョン

2025年〜2026年の市場変容（「デジタルの手間代行（Digital Labor Agency）」「五感リアル体験」）に最適化した、女性向けセルフケア投資の「信頼のインフラ」を構築する。

- **核心的価値**: 効率化（AI診断・多軸検索）と情緒（シネマティック動画・体験型UI）の統合。
- **ポジショニング**: 食べログ型（目的達成）× Netflix型（受動的発見）のハイブリッド。

## 2. 実装済みコンポーネント (src/app/test13/page.tsx)

現在のプロトタイプには以下の主要機能が実装されています。

- **シネマティック・ヒーロー**: 90秒で価値を伝える高品質なビジュアルとPREP法キャッチコピー。
- **10クラスター・ダイナミックUI**:
  - ユーザー属性（1〜10）に応じたフォント、配色、ナビゲーションの動的変更ロジック。
  - 特に「クラスター10（シニア層）」への高視認性対応を強化。
- **運営メディア統合 (Ecosystem)**:
  - 「イケ男ラボ」「イケ女ラボ」「スイートホテル」の最新記事セクション。
  - サムネイルの出し分け（Pulp Fictionモデル）シミュレーション機能。
- **ハイブリッド・インターフェース**:
  - Netflix風の行単位レコメンド。
  - Tablog（食べログ）風の多軸条件検索パネル。
- **AI診断導線**: 心理的障壁を解消する成約トリガーとしての診断ボタン。

## 3. 技術スタック

- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide-React
- **SEO/AIO**: JSON-LD 構造化データの自動生成、PREP法に基づいたコンテンツ配置。

## 4. クラスター定義 (10 Personas)

開発を再開する際は、以下の定義をUI/UXの基準としてください。

1. **プレ社会人**: ASMR、短尺動画、ビビッド
2. **若手シングル**: 平成女児カルチャー、没入感
3. **中堅シングル**: 平穏ミュート、専門家監修
4. **ベテランシングル**: Re良質消費、エビデンス
5. **ヤング夫婦**: お金のペアスタイル、共有
6. **ミドル夫婦**: 持たない住まい、本物志向
7. **乳幼児ママ**: 家事速攻完了、Q&A
8. **児童ママ**: 異作業の一本化、ピラーページ
9. **青年期ママ**: 楽外出、モバイル・位置情報
10. **セカンドライフ**: デジタルスキル65、高視認性

## 5. 未実装・将来の拡張計画

- **メディア情報の動的取得**: 現在はモックデータのメディア情報をAPI連携に切り替える。
- **個別店舗の詳細連携**: 検索結果から各支店ページへのシームレスな遷移。
- **AI診断のアルゴリズム実装**: ユーザーの行動ログに基づいたリアルなレコメンド。
- **UGC（口コミ・写メ）のDB連携**: キャストが投稿する日記やつぶやきのリアルタイム反映。

2. 要件定義書（チャット履歴の「次世代型女性向けコンテンツハブ・プラットフォーム 要件定義書」）の内容を再度読み込む。
3. `src/app/test13/HANDOVER.md`（本ファイル）の各項目に沿って機能を拡張する。
