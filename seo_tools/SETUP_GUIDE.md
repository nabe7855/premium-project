# SEO分析環境 セットアップガイド

このツール群は、Google Search Consoleのデータを自動取得し、分析するための基盤です。
開始するには、Google Cloudでの設定が必要です。

## 1. Google Cloud プロジェクトの準備

1. [Google Cloud Console](https://console.cloud.google.com/) にアクセスします。
2. 新しいプロジェクトを作成するか、既存のプロジェクトを選択します。
3. **「APIとサービス」 > 「ライブラリ」** へ移動します。
4. **「Google Search Console API」** を検索し、**「有効にする」** をクリックします。

## 2. 認証情報の作成（サービスアカウント）

1. **「APIとサービス」 > 「認証情報」** へ移動します。
2. **「認証情報を作成」 > 「サービスアカウント」** を選択します。
3. 名前（例: `seo-analyzer`）を入力し、作成します。
4. 役割の付与はスキップ（完了）してOKです。
5. 作成したサービスアカウントのメールアドレス（例: `seo-analyzer@...iam.gserviceaccount.com`）をコピーしておきます。

## 3. JSON鍵の発行

1. 作成したサービスアカウントの詳細画面を開き、**「キー」** タブへ移動します。
2. **「鍵を追加」 > 「新しい鍵を作成」** を選択し、**JSON** 形式で作成します。
3. ダウンロードされたJSONファイルを、以下の名前に変更して保存してください：
   - パス: `c:\Users\nabe7\Documents\Projects\premium-project\seo_tools\credentials.json`
   - **※このファイルは絶対に公開しないでください（.gitignoreに追加済みです）**

## 4. Search Consoleでの権限付与

1. [Google Search Console](https://search.google.com/search-console) にアクセスします。
2. 設定したいサイト（プロパティ）を選択します。
3. **「設定」 > 「ユーザーと権限」** へ移動します。
4. **「ユーザーを追加」** をクリックします。
5. 先ほどコピーした **サービスアカウントのメールアドレス** を入力し、権限を **「フル」** または **「オーナー」** に設定して追加します。
