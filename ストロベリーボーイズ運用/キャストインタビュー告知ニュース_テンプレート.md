# キャストインタビュー告知ニュース テンプレート

連載インタビュー公開時、**ニュースページに掲載する短い告知記事**のテンプレートです。
ニュース側は「告知＋導線」に徹し、SEOの主役（リッチコンテンツ）はインタビュー記事側に集約させる設計になっています。

---

## 設計思想（最初に1回だけ読めばOK）

1. **ニュースは「フィーダー」、インタビューは「ハブ」**
   ニュース記事は短く、本体（インタビュー）へリンク権威を流すための入口にする。長文化させない。
2. **キーワードを微妙にズラす**
   インタビュー本体が `塩顔セラピスト サイ 福岡` を狙うなら、ニュース側は `インタビュー公開 サイ ストロベリーボーイズ` のように"公開・お知らせ"系の語を入れて住み分け（自社内カニバリ防止）。
3. **構造化データを使い分け**
   インタビュー本体：`Article` + `FAQPage`（深い情報）
   ニュース告知：`NewsArticle`（鮮度シグナル）
4. **必ず3つの導線を仕込む**
   ①インタビュー本文へ ②キャスト個別プロフィール／予約へ ③過去インタビュー一覧へ

## 1. SEOメタ情報テンプレート

### `<title>` （28〜32字目安）
```
【インタビュー公開】{{cast_face_type}}{{cast_career_tag}}・{{cast_name}}の素顔｜ストロベリーボーイズ
```

### `<meta name="description">` （120〜140字）
```
{{area_jp}}エリアのキャスト「{{cast_name}}」の連載インタビューvol.{{vol_number}}を公開しました。{{cast_height}}・{{cast_face_type}}、{{cast_specialty}}が得意な{{cast_name}}の素顔に、女性向け風俗ライター・イトウが迫ります。「{{interview_title_short}}」全文公開中。
```

### 推奨URL
```
/news/{{YYYY}}/{{MM}}/{{cast_name_en}}-interview-vol{{vol_number}}/
```

### OGP
```html
<meta property="og:title" content="【vol.{{vol_number}}公開】{{cast_face_type}}{{cast_career_tag}}・{{cast_name}}に聞いた、{{interview_title_short}}">
<meta property="og:description" content="{{area_jp}}・{{cast_name}}の素顔に迫る連載インタビュー、本日公開。プロフィールでは伝わらない空気感を読む。">
<meta property="og:image" content="{{ogp_image_url}}">
<meta property="og:type" content="article">
<meta name="twitter:card" content="summary_large_image">
```

## 5. 公開前チェックリスト

タイトル・メタ系
- [ ] titleは32字以内、「インタビュー公開」「キャスト名」を含む
- [ ] meta descriptionは120〜140字、不自然なキーワード詰め込みなし
- [ ] URLは英小文字＋ハイフンのみ（日本語URL不使用）

画像
- [ ] サムネイル画像は16:9（推奨1920×1080）
- [ ] OGP画像は1200×630で別途用意
- [ ] 画像ファイル名は記述的に（例：`sai-interview-vol1-thumb.jpg`）
- [ ] `alt`属性に「{{cast_name}}インタビュー告知サムネイル」を設定

リンク・導線
- [ ] インタビュー本文への内部リンクが本文中に**最低2回**ある
- [ ] キャストプロフィール／予約ページへのリンクがある
- [ ] 過去インタビュー一覧へのリンクがある
- [ ] CTAボタン（「全文を読む」）が視認性高く設置されている

構造化データ
- [ ] `NewsArticle`のJSON-LDが`<head>`に入っている
- [ ] `mainEntityOfPage`がインタビュー本体URLを指している（重要：これでリンク権威が本体に流れる）
- [ ] [Google リッチリザルトテスト](https://search.google.com/test/rich-results)でエラーなし

公開後
- [ ] Google Search Consoleで該当URLをインデックス申請
- [ ] サイトマップXMLを更新
- [ ] SNS（X / Instagram）にOGP付きでシェア
- [ ] 1週間後：Search Consoleで表示回数とCTRを確認

## 6. 運用Tips（連載を回す時の地味コツ）

- **告知記事の本文は400〜600字に抑える**。長くするとインタビュー本体とSEOがカニバる
- **公開タイミングは平日18-21時 or 土日午前**。女性ユーザーがスマホを触る時間帯
- **過去のニュース告知も育てる**：vol.2公開時に「vol.1記事を読む」リンクを追加すると、古い告知記事もリフレッシュされる
- **`/interview/`ハブページが3本以上溜まったら作る**。それ未満だと薄いページになりSEO逆効果
- **キャスト本人のSNSにもシェアしてもらう設計**にする（コピペ用文面を用意してあげると協力率が上がる）
