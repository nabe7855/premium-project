# 料金セクション 切り出し素材

カタログシート（番号付きの部品一覧画像）から各部品を1枚ずつのPNGに切り分けました。ファイル名の番号は、元カタログ画像のラベル番号（01〜）と対応しています。

## フォルダ構成

- `A_お得なイベントコース/` … 元画像「10_47_37 (1)」由来。部品 01〜24（＋余白）
- `B_ロングコース/` … 元画像「11_17_17 (1)」由来。部品 01〜28
- `C_スタンダードコース/` … 元画像「11_17_23 (1)」由来。部品 01〜27
- `高解像度価格バー/` … 2172×724 の価格バー単体画像（12枚）を透過処理したもの

※ カタログシートは4枚ありましたが、うち1枚（11_17_31）は10_47_37と完全に同一だったため、ユニークな3種を処理しています。

## 背景透過の方針（プロ向け・レイヤー重ね想定）

- ロゴ・王冠・イチゴ装飾・区切り線・タブボタン・CTA・価格バー・アイコン・テキスト等
  → 外側の白背景のみ透過。ロゴ内部の白などは保持。
- 番号・ラベル文字は各セルから除去済み（部品の絵柄のみ）。
- ヒーロー背景写真（05）は、透過すると穴が空くため矩形の不透明PNGのまま。
- 白／クリームの余白スウォッチは、透過すると消えるため不透明のまま。
