# n8n ローカル環境構築 ＆ セットアップ手順

この手順書では、**あなたの PC (Windows) に、10分で自動化の司令塔「n8n」を無料インストール** し、運用システムを動かす基盤を作ります。

## 1. 事前準備
以下のうち、いずれか 1 つが必要です。

### **【方法 A】Docker を使用（おすすめ）**
最も推奨される方法です。他のアプリに影響せず、簡単に「消したり作り直したり」できます。
1.  [Docker Desktop](https://www.docker.com/products/docker-desktop/) をダウンロードしてインストールします。
2.  パソコンを再起動します。

---

## 2. インストール実行手順

### **ステップ 1: n8n の起動**
PowerShell または コマンドプロンプトを開き、以下のコマンドを 1 行コピーして実行してください。

```powershell
docker run -it --rm --name n8n -p 5678:5678 -v n8n_data:/home/node/.n8n n8nio/n8n
```

### **ステップ 2: ブラウザでアクセス**
1.  コマンド実行後、しばらく待つとログが止まります。
2.  ブラウザで `http://localhost:5678` を開きます。
3.  初期設定画面（メールアドレス・パスワードの設定）が出るので、案内に従って入力します。

---

## 3. 基本設定 ＆ 動作確認

### **① 言語設定（任意）**
*   右上の設定（歯車アイコン）から「Language」を「Japanese」に変更できます。

### **② 最初のテスト**
1.  「Create your first workflow」をクリック。
2.  Node の追加（＋ボタン）から `Schedule` (タイマー的役割) を追加。
3.  次に `Discord` や `Line` (通知用) または `Google Sheets` を繋いでみる練習をします。

---

## 4. 注意事項
*   **稼働時間**: PC の電源を切ると `localhost:5678` にもアクセスできず、自動化も止まります。
*   **データの保存**: `-v n8n_data:/home/node/.n8n` という記述のおかげで、再起動しても作成したワークフローは保存されます。

---

## 5. 次のステップ
n8n が立ち上がったら、次は以下の API キーを取得し、n8n に登録（Credential といいます）します。
*   **Perplexity API Key**
*   **xAI (Grok) API Key** (または X API)
*   **OpenAI / Claude API Key** (執筆用)

セットアップが完了したら、`strategy/prompts.md` を参考に、最初の「トレンド取得フロー」を作っていきましょう！
