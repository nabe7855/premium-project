## 1. 現状分析と戦略の全体像

### 1.1 現状

- SNSアカウント：全プラットフォーム未開設
- 運用体制：代表1人 + キャスト（ゆくゆく2〜3人体制へ）
- サイト：Next.js製の本格的なWebアプリケーション（店舗ページ、SweetStay、イケジョラボ、イケオラボ）が構築済み
- SEO基盤：MEO計画、構造化データ計画、ハブページ設計が策定済みだが、外部からの流入導線（SNS）が未整備
- 競合：アンモモンプートア（福岡）等が先行。ただしSNS活用は業界全体で未成熟

### 1.2 戦略の核心：「SEOを上げるためにSNSを使う」

SNSのリンクはnofollow属性のため直接的なSEO効果はない。しかし、SNSは以下の3つの間接効果でSEOを強力にブーストする。

**① 指名検索の増加** → Googleは「ストロベリーボーイズ」の検索ボリューム増加をブランド権威性のシグナルとして評価する

**② サイテーション（言及）の獲得** → SNS上で店名・サービス名が言及されるほど、Googleのエンティティ認識が強化される

**③ 自然被リンクの誘発** → SNSでバズったコンテンツを見たブロガー・メディアが自サイトからリンクを貼る

つまり、SNSの目的は「フォロワー数を増やすこと」ではなく、**「ストロベリーボーイズ」という固有名詞をWeb上に大量に生成し、指名検索とサイテーションを爆増させること**である。

## 2. SNS × SEO の連動メカニズム

### 2.1 コンテンツ循環フロー

```
[TikTok/YouTube Shorts]  認知獲得（バイラル）
         ↓
[Instagram/X]  フォロー → エンゲージメント → 指名検索
         ↓
[YouTube長尺/Podcast]  信頼構築 → 専門性認知
         ↓
[サイト記事へ誘導]  SweetStay / イケジョラボ / 店舗ページ
         ↓
[記事がSEO上位化]  → 自然検索からの新規流入
         ↓
[SNSフォロー/シェア]  → さらなるサイテーション増加
         ↓
（循環）
```

### 2.2 各SNSのSEOへの貢献方法

**YouTube → SEO直結度：最高**
- YouTube動画はGoogle検索結果に直接表示される
- 「女性用風俗 福岡」で動画枠を獲得すれば、サイトが2位でも実質的にトップ表示
- 動画説明文にサイトURLを記載 → 参照トラフィック増
- YouTubeはGoogle傘下なので、Googleのエンティティグラフに直接寄与

**TikTok → SEO直結度：中（認知爆発力は最高）**
- TikTokはGoogleインデックスに含まれないが、バイラルによる指名検索爆増が起こる
- 「ストロベリーボーイズって何？」→ Google検索 → SEO指名検索数が増加

**Instagram → SEO直結度：中**
- Instagramプロフィール・投稿はGoogleにインデックスされる
- ハッシュタグ戦略で「女性用風俗 横浜」等のキーワードを含む投稿が検索結果に出現
- ストーリーズ/リールからのサイト誘導 → 参照トラフィック増

**X（Twitter） → SEO直結度：中高**
- ツイートはGoogleにインデックスされやすい
- 拡散力が最も高く、サイテーション獲得に最適
- ただし、女性用風俗関連はアカウント凍結リスクが高い（後述）

**Podcast → SEO直結度：低〜中**
- 直接的なSEO効果は薄いが、ブランド認知・指名検索に貢献
- Show notesにサイトリンクを記載 → 被リンク効果
- Apple Podcast/SpotifyのページはGoogleにインデックスされる

## 3. アカウント設計とブランディング体系

### 3.1 アカウント構成（推奨）

1人運用を考慮し、**1ブランド × 複数用途**のシンプルな構成にする。メディアごとにアカウントを分けると管理が破綻するため、統一アカウントで運用し、コンテンツカテゴリーで使い分ける。

| プラットフォーム | アカウント名案 | 運用方針 |
|---|---|---|
| YouTube | ストロベリーボーイズ【公式】 | メインチャンネル1本。再生リストでカテゴリ分け |
| TikTok | @strawberry_boys_official | 統一アカウント。ショート動画に集中 |
| Instagram | @strawberry_boys_official | 統一アカウント。ストーリーズ活用 |
| X | @strawberry_boys | メイン情報発信。サブとしてキャスト個人アカウントも検討 |
| Podcast | ストロベリーボーイズの夜話 | Spotify/Apple配信。週1更新 |

**補足：キャストの個人アカウント運用**
- キャストが自分のInstagram/TikTokで「仕事風景」「日常」を発信するのは強力な集客手段
- ただし1人運用フェーズでは管理しきれないため、Phase 2以降で導入
- キャスト個人アカウントからは必ず公式アカウントをタグ付けするルールを設ける

### 3.2 プロフィール設計のSEOポイント

全プラットフォーム共通で以下を必ず含める：

- **サービス名**：ストロベリーボーイズ（必ずカタカナで統一）
- **地域名**：横浜・福岡（地域SEOに貢献）
- **カテゴリ明示**：女性のための出張ホストサービス（「女性用風俗」はプラットフォームのポリシー抵触リスクがあるため、やわらかい表現を使う）
- **サイトURL**：必ず記載（参照トラフィック + サイテーション）
- **統一ハッシュタグ**：#ストロベリーボーイズ（ブランドハッシュタグ）

### 3.3 プラットフォーム別の表現ガイドライン

Xのアカウント凍結リスクを踏まえ、全プラットフォームで「ソフトな表現」を基本とする。

**使ってよい表現例：**
- 女性専用リラクゼーション
- 出張ホスト
- 女性のための癒やしサービス
- パートナーシップサポート
- セルフケア、ウェルネス

**避けるべき表現（凍結リスク）：**
- 直接的な性的表現・ワード
- 「風俗」の単語そのもの（記事リンクや動画説明文では使用可、プロフィールでは避ける）
- 連続した宣伝的投稿

   - 男性向け自己改善コンテンツ
   - 「女性が本当に喜ぶデートプラン」
   - キャスト募集につながるブランディング
   - 形式：5〜10分。キャスト出演（顔出し可の人）

4. **【ストベリTV】** ← ブランド認知
   - 店舗紹介、キャスト紹介、お客様の声（匿名）
   - 業界Q&A「女性用風俗ってどんなところ？」
   - 形式：3〜8分。FAQ形式

**YouTube Shorts（TikTok並行投稿用）：**

- TikTok/Instagram Reelsにもそのまま転用

- チャプター機能で構造化（Googleの動画リッチスニペット対応）
- 字幕（CC）を必ず追加（Googleがテキストとして認識）

**投稿頻度（1人運用フェーズ）：**
- 長尺動画：週1本
- Shorts：週3本（TikTok/Reelsと並行）

### 4.4 X（Twitter）（拡散力とリアルタイム性）

**運用の前提：凍結リスクへの対策**

   - サイテーション獲得が目的

3. **店舗・キャスト情報（20%）**
   - 出勤情報、新人紹介
   - キャンペーン告知
   - CV（予約・問い合わせ）が目的

**投稿頻度：**
- 1日3〜5ツイート（うちリンク付きは2回まで）

## 7. SEO直結施策：指名検索とサイテーション戦略

### 7.1 指名検索を増やす具体的アクション

**① 動画の冒頭と末尾で店名を必ず言う**
- 「ストロベリーボーイズの〇〇です」で始まる
- 「詳しくはストロベリーボーイズで検索してね」で終わる
- これにより視聴者がGoogle検索する → 指名検索数が増加

**② TikTokのプロフィールにサイトURLを記載し「続きはプロフィールのリンクから」と誘導**
- TikTokは外部リンクをキャプションに貼れないため、プロフィールリンクに誘導
- 視聴者がURLではなく「ストロベリーボーイズ」で検索するパターンも多い → 指名検索増

**③ Xでバズ投稿 → 「何これ？」→ Google検索の流れを作る**
- 共感ツイートがバズった際、プロフィールに誘導文を用意
- 興味を持った人が店名で検索 → 指名検索増

**④ Podcastで毎回「検索してね」を言う**
- 音声メディアの特性上、視聴者はURLを打てない → 自然に指名検索が発生

### 7.2 サイテーション（言及）を獲得する方法

**① 引用したくなるデータ・ランキングを作る**
- 「福岡ラブホテル満足度ランキング2026」のような独自データ
- メディア・ブロガーが引用する → サイテーション + 被リンク

**② 業界レポートを定期発行**
- 「女性用風俗業界レポート」のような資料をPDFで公開
- 引用元として「ストロベリーボーイズ調べ」と記載される → サイテーション

**③ SNSでのUGC（ユーザー生成コンテンツ）促進**
- 「#ストベリ体験談」のようなハッシュタグキャンペーン
- ユーザーが体験談を投稿 → サイテーション大量発生

**④ プレスリリース活用**
- サービス開始時、新店舗オープン時にPR TIMESでプレスリリース配信
- ニュースサイトに転載される → 被リンク + サイテーション

### 7.3 被リンク獲得戦略

**① SweetStayの「ラブホテルデータベース」を被リンク対象に**
- 地域のラブホ情報をまとめた権威性の高いデータベースを構築
- デート系ブログ、旅行サイトからリンクを獲得

**② イケジョラボの教育コンテンツを被リンク対象に**
- セルフケア、フェムテック、セクシャルウェルネスの教育記事
- 健康系メディア、フェムテック企業からの引用・リンク

**③ YouTube動画からの被リンク**
- 他のYouTuberが「参考にした動画」として紹介 → 概要欄にリンク

## 11. リスク管理とコンプライアンス

### 11.1 プラットフォーム別リスク

**X（Twitter）— 凍結リスク：高**
- 対策：ソフトな表現の徹底、リンク連投回避、複数アカウント分散禁止
- 万が一の凍結に備え、他プラットフォームでの集客基盤を先に構築

**TikTok — コンテンツ削除リスク：中**
- 対策：コミュニティガイドラインを熟読し、直接的な表現を避ける
- 「教育コンテンツ」「情報提供」のフレーミングで投稿

**YouTube — 年齢制限リスク：中**
- 対策：年齢制限がかかっても検索表示される動画SEO設計
- コンテンツは「教育・情報」カテゴリに設定

**Instagram — アカウント制限リスク：低〜中**
- 対策：過度な外部リンク誘導を避ける、DM自動送信ツールは使わない

### 11.2 法的コンプライアンス

- 風営法に準拠した表現を使用
- キャストのプライバシー保護（顔出しは本人の同意を書面で取得）
- お客様の声・体験談は匿名化を徹底
- 18禁コンテンツのプラットフォーム規約遵守
- 景品表示法への配慮（誇大広告の回避）

### 11.3 炎上対策

- SNS投稿は公開前にチェックリストを通す
- ネガティブコメントへの対応マニュアルを作成
- 危機対応フロー：発見 → 一時非公開 → 対応方針決定 → 公式回答

## まとめ：この戦略がSEO1位に到達するロジック

```
SNS認知拡大
  → 指名検索「ストロベリーボーイズ」の増加
    → Googleがブランドエンティティとして認識
      → ブランドキーワードでの順位上昇

SNSコンテンツ → サイト記事への送客
  → サイトのトラフィック増加・滞在時間向上
    → Googleがサイトの質を高く評価
      → 一般キーワード「女性用風俗 横浜」等での順位上昇

SNSでの話題化 → メディア・ブロガーからの被リンク
  → ドメインオーソリティの上昇
    → 全キーワードでの順位底上げ

SweetStay/イケジョラボのコンテンツ量産
  → ロングテールキーワードの大量獲得
    → サイト全体のトピックオーソリティ上昇
      → ヘッドキーワード「女性用風俗 〇〇」での順位上昇
```

最も重要なのは **「SNSで認知を取る → 指名検索を増やす → SEOが上がる」** というシンプルなループを、6ヶ月間ひたすら回し続けること。

1人でも、TikTok × Instagram × Xの3本柱を毎日回すだけで、このループは確実に動き始める。
